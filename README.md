CLIHelper
=========

Easy to use CLI API in JAVA

## Installation

Checkout the source code:

    git clone https://github.com/JM-Lab/CLIHelper.git
    cd CLIHelper
    mvn install

## Usage
Set up Maven POM file :

    (...)
    <dependency>
			<groupId>jmlab</groupId>
			<artifactId>jmlab-cli</artifactId>
			<version>1.0.0</version>
	</dependency>
    (...)
